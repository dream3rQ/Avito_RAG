# Avito Data Science Bootcamp 2026: Тестовое задание 🚀

Решение задачи информационного поиска (Information Retrieval) по базе статей справки Авито. 

**🏆 Финальная метрика на скрытом тесте (MAP@10): 0.480**

## 🛠 Архитектура решения (Retrieve & Rerank)
Пайплайн построен с использованием локальных open-source моделей и разделен на два этапа для баланса между скоростью инференса и точностью:
1. **Hybrid Retrieval:** Лексический поиск (тюнованный `BM25` с лемматизацией) + Семантический поиск (`BAAI/bge-m3`), объединенные через взвешенный алгоритм `RRF`.
2. **Cross-Encoder Reranking:** Финальное ранжирование топ-кандидатов тяжелой нейросетью `BAAI/bge-reranker-v2-m3` с использованием батчевой обработки для 100% утилизации GPU.

## 📂 Структура репозитория

  * Хардверные оптимизации (борьба с OOM и CPU bottleneck);
